# MemoraAI 🧠

MemoraAI is an AI-powered personal memory assistant designed to help users organize their daily life, remember important information, and manage tasks efficiently.

## Features

- 🤖 AI Companion
- 📝 Notes Management
- ✅ Task Management
- 🧠 Memory Storage
- 📷 Photo Memories
- 💊 Medication Tracking
- 🚨 Emergency SOS
- 👤 User Profile
- ☀️ Daily Briefing Dashboard

## Technology Stack

- React
- TypeScript
- Node.js
- Google Gemini AI
- Replit
- GitHub

## Installation

```bash
git clone https://github.com/eprem02/Memora-AI.git
cd Memora-AI
npm install
